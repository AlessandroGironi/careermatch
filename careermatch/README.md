# CareerMatch

CareerMatch is a Docker-ready web app that compares a CV with a LinkedIn job post and generates a fit report (score, strengths, gaps, and suggestions).

## Requirements
- Docker
- Docker Compose

## Setup

### 1) Download
- Clone the repo (or download ZIP from GitHub)
- Enter the project folder

### 2) Configure environment
Create your `.env` file from the example:

```bash
cp .env.example .env
